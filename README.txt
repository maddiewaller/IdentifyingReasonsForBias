Code to replicate experiments in the submitted AAAI 2024 paper:
Identifying Reasons for Bias: An Argumentation-Based Approach

Run using
- Python 3.10
- Conda 22.9.0

Conda environment specified in requirements.txt

__main__.py runs the experiments for the paper
Which experiments to run are specified as input:
- 'adult' for the Adult dataset
- 'bank' for the Bank Marketing dataset
- 'synthetic' for the adapted Adult dataset with added bias
- 'example' for the simple example in the paper

Results of experiments, matching those in the paper, are stored in the results folder.

All code is stored in the modules folder.
